# My Infrastructure

My Infrastructure is a smart tool designed to help users manage their infrastructure effectively. It offers various features to streamline processes and enhance productivity.

## Features
- Infrastructure management
- Automation tools
- Monitoring capabilities

## Tech Stack
- Python
- Flask
- Docker

## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/svinia2/my-infrastructure.git
   ```
2. Navigate to the project directory:
   ```bash
   cd my-infrastructure
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```
